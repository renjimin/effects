列出了所有在此版本的 jQuery 中新增或修改的 API。
jQuery 1.4 Release Notes.

Also in: Traversing > Miscellaneous Traversing

.add()

添加元素到匹配的元素集合。

Also in: Attributes | Manipulation > Class Attribute | CSS

.addClass()

为每个匹配的元素添加指定的样式类名

Also in: Manipulation > DOM Insertion, Outside

.after()

在元素集合中匹配的每个元素后面插入参数所指定的内容。

Also in: Manipulation > DOM Insertion, Inside

.append()

根据参数设定在每个匹配元素里面的末尾处插入内容。

Also in: Manipulation > DOM Insertion, Outside

.before()

根据参数设定在匹配元素的前面插入内容（译者注：外部插入）。

Also in: Events > Event Handler Attachment

.bind()

为一个元素绑定一个事件处理程序。

Also in: Effects > Custom | Data | Utilities

.clearQueue()

从列队中移除所有未执行的项。

Also in: Traversing > Tree Traversal

.closest()

从元素本身开始，在DOM 树上逐级向上级元素匹配，并返回最先匹配的祖先元素。

Also in: CSS | Manipulation > Style Properties

.css()

获取匹配元素集合中的第一个元素的样式属性的值 或 设置每个匹配元素的一个或多个CSS属性。

Also in: Data | Miscellaneous > Data Storage

.data()

在匹配元素上存储任意相关数据 或 返回匹配的元素集合中的第一个元素的给定名称的数据存储的值。

Also in: Effects > Custom

.delay()

设置一个延时来推迟执行队列中后续的项的执行。

Also in: Manipulation > DOM Removal

.detach()

从DOM中去掉所有匹配的元素。

Also in: Traversing > Filtering

.filter()

筛选元素集合中匹配表达式或传递函数的测试的那些元素集合。

Also in: Traversing > Filtering

.first()

获取匹配元素集合中第一个元素。

Also in: Events > Keyboard Events | Events > Mouse Events

.focusin()

将一个事件函数绑定到"focusin" 事件。

Also in: Events > Keyboard Events | Events > Mouse Events

.focusout()

将一个事件函数绑定到"focusout" 事件。

Also in: Traversing > Filtering

.has()

筛选匹配元素集合中的那些有相匹配的选择器或DOM元素的后代元素。

Also in: Attributes | Manipulation > DOM Insertion, Inside

.html()

从集合的第一个匹配元素中获取HTML内容 或 设置每一个匹配元素的html内容。

Also in: Miscellaneous > DOM Element Methods

.index()

从匹配的元素中搜索给定元素的索引值，从0开始计数。

Also in: Core

jQuery()

返回匹配的元素集合 无论是通过 在DOM的基础上传递的参数还是创建一个HTML字符串。

Also in: Utilities

jQuery.contains()

Check to see if a DOM element is a descendant of another DOM element.

Also in: Data | Utilities

jQuery.data()

存储任意数据到指定的元素 并且/或者 返回设置的值。

Also in: Utilities

jQuery.isEmptyObject()

检查对象是否为空（不包含任何属性）。

Also in: Utilities

jQuery.isPlainObject()

测试对象是否是纯粹的对象（通过 "{}" 或者 "new Object" 创建的）

Also in: Utilities

jQuery.noop()

一个空函数

Also in: Miscellaneous > Collection Manipulation | Forms | Ajax > Helper Functions

jQuery.param()

创建一个序列化的数组或对象，适用于一个URL 地址查询字符串或Ajax请求。

Also in: Events > Event Handler Attachment | Utilities

jQuery.proxy()

接受一个函数，然后返回一个新函数，并且这个新函数始终保持了特定的上下文语境。

Also in: Traversing > Filtering

.last()

获取匹配元素集合中最后一个元素。

Also in: Traversing > Tree Traversal

.nextUntil()

通过选择器，DOM节点，或jQuery对象得到每个元素接下来的所有的兄弟元素，但不包括匹配的元素。

Also in: Traversing > Filtering | Traversing > Miscellaneous Traversing

.not()

删除匹配的元素集合中元素。

Also in: CSS | Offset | Manipulation > Style Properties

.offset()

在匹配的元素集合中，获取的第一个元素的当前坐标， 或设置每一个元素的坐标， 坐标相对于文档。

Also in: Traversing > Tree Traversal

.parentsUntil()

查找当前元素的所有的前辈元素，直到遇到选择器匹配的元素为止，不包括那个匹配到的元素。

Also in: Manipulation > DOM Insertion, Inside

.prepend()

将参数内容插入到每个匹配元素的前面（元素内部）。

Also in: Traversing > Tree Traversal

.prevUntil()

获得集合中每个匹配元素的所有前面的兄弟元素，选择性筛选的选择器。

Also in: Attributes | Manipulation > General Attributes

.removeAttr()

为匹配的元素集合中的每个元素中移除一个属性。

Also in: Attributes | Manipulation > Class Attribute | CSS

.removeClass()

移除集合中每个匹配元素上一个，多个或全部样式。

Also in: Manipulation > DOM Replacement

.replaceWith()

用提供的内容替换集合中所有匹配的元素并且返回被删除元素的集合。

Also in: Manipulation > DOM Insertion, Inside

.text()

得到匹配元素集合中每个元素的文本内容结合，包括他们的后代，或设置匹配元素集合中每个元素的文本内容为指定的文本内容。

Also in: Miscellaneous > DOM Element Methods

.toArray()

返回一个包含jQuery对象集合中的所有DOM元素的数组。

Also in: Attributes | Manipulation > Class Attribute | CSS

.toggleClass()

在匹配的元素集合中的每个元素上添加或删除一个或多个样式类,取决于这个样式类是否存在或价值切换属性。即：如果存在（不存在）就删除（添加）一个类。

Also in: Manipulation > DOM Insertion, Around | Manipulation > DOM Removal

.unwrap()

将匹配元素集合的父级元素删除，保留自身（和兄弟元素，如果存在）在原来的位置。

Also in: Attributes | Forms | Manipulation > General Attributes

.val()

获取匹配的元素集合中第一个元素的当前值或设置匹配的元素集合中每个元素的值。

Also in: Manipulation > DOM Insertion, Around

.wrap()

在每个匹配的元素外层包上一个html元素。

Also in: Manipulation > DOM Insertion, Around

.wrapAll()

在所有匹配元素外面包一层HTML结构。

Also in: Manipulation > DOM Insertion, Around

.wrapInner()

在匹配元素里的内容外包一层结构。
