分类：事件

当用户与浏览器进行交互时这些方法被用于注册行为生效， 并进一步处理这些注册行为。

Also in: Events > Event Handler Attachment

.bind()

为一个元素绑定一个事件处理程序。

Also in: Events > Form Events | Forms

.blur()

为 "blur" 事件绑定一个处理器，或者触发元素上的 "blur" 事件。

Also in: Events > Form Events | Forms

.change()

为 "change" 事件绑定一个处理函数，或者触发元素上的 "change" 事件。

Also in: Events > Mouse Events

.click()

为 "click" 事件绑定一个处理器，或者触发元素上的 "click" 事件。

Also in: Events > Mouse Events

.dblclick()

为 "dblclick" 事件绑定一个处理函数，或者触发元素上的 "dblclick" 事件。

Also in: Events > Event Handler Attachment

.delegate()

为所有匹配选择器（selector参数）的元素绑定一个处理器到一个或多个事件，现在或将来， 基于根元素的一个特定的组

Also in: Events > Event Handler Attachment

.die()

从元素中删除先前用.live()绑定的所有事件。

Also in: Events > Browser Events

.error()

绑定一个事件处理器到"error" JavaScript事件上。

Also in: Events > Event Object

event.currentTarget

在事件冒泡过程中的当前DOM元素。

Also in: Events > Event Object

event.data

一个可选的数据对象传递给一个事件方法，当当前正在执行的处理程序被绑定。

Also in: Events > Event Object

event.delegateTarget

绑定了当前正在调用jQuery 事件处理器的元素。

Also in: Events > Event Object

event.isDefaultPrevented()

根据事件对象中是否调用过 event.preventDefault() 方法,来返回一个布尔值。

Also in: Events > Event Object

event.isImmediatePropagationStopped()

根据事件对象中是否调用过 event.stopImmediatePropagation() 方法,来返回一个布尔值。

Also in: Events > Event Object

event.isPropagationStopped()

根据事件对象中是否调用过 event.stopPropagation() 方法,来返回一个布尔值。

Also in: Events > Event Object

event.metaKey

表示事件触发时哪个Meta键被按下。

Also in: Events > Event Object

event.namespace

当事件被触发时此属性包含指定的命名空间。

Also in: Events > Event Object

event.pageX

鼠标相对于文档的左边缘的位置。

Also in: Events > Event Object

event.pageY

鼠标相对于文档的顶部边缘的位置。

Also in: Events > Event Object

event.preventDefault()

如果调用这个方法，默认事件行为将不再触发。

Also in: Events > Event Object

event.relatedTarget

在事件中涉及的其它任何DOM元素。

Also in: Events > Event Object

event.result

事件被触发的一个事件处理程序的最后返回值，除非值是 undefined。

Also in: Events > Event Object

event.stopImmediatePropagation()

阻止剩余的事件处理函数执行并且防止事件冒泡到DOM树上。

Also in: Events > Event Object

event.stopPropagation()

防止事件冒泡到DOM树上，也就是不触发的任何前辈元素上的事件处理函数。

Also in: Events > Event Object

event.target

触发事件的DOM元素。

Also in: Events > Event Object

event.timeStamp

这个属性返回事件触发时距离1970年1月1日的毫秒数。

Also in: Events > Event Object

event.type

描述事件的性质。

Also in: Events > Event Object

event.which

针对键盘和鼠标事件，这个属性能确定你到底按的是哪个键。

Also in: Events > Form Events | Forms

Also in: Events > Keyboard Events | Events > Mouse Events

.focusin()

Also in: Events > Keyboard Events | Events > Mouse Events

Also in: Events > Mouse Events

.hover()

Also in: Events > Event Handler Attachment | Utilities
