分类：操作

此部分中所有的方法是一些操作DOM的方式。 其中几个方法是简单的改变一个元素的一个属性（也列在属性分类中）， 其他几个设置元素的样式属性的方法 （也列在CSS 分类中）。 还有一些方法修改整个元素（或一组元素）本身—插入，复制，删除，等等。 所有这些方法被归类为为“setters”（设置器），因为他们改变属性的值。 有些方法 —如.attr(), .html(), and .val() — 还充当“getters”（获取器），从DOM元素检索信息，供以后使用。

Also in: Attributes | Manipulation > Class Attribute | CSS

.addClass()

为每个匹配的元素添加指定的样式类名

Also in: Manipulation > DOM Insertion, Outside

.after()

在元素集合中匹配的每个元素后面插入参数所指定的内容。

Also in: Manipulation > DOM Insertion, Inside

.append()

根据参数设定在每个匹配元素里面的末尾处插入内容。

Also in: Manipulation > DOM Insertion, Inside

.appendTo()

将匹配的元素插入到目标元素的最后面（译者注：内部插入）。

Also in: Attributes | Manipulation > General Attributes

.attr()

获取匹配的元素集合中的第一个元素的属性的值 或 设置每一个匹配元素的一个或多个属性。

Also in: Manipulation > DOM Insertion, Outside

.before()

根据参数设定在匹配元素的前面插入内容（译者注：外部插入）。

Also in: Manipulation > Copying

.clone()

创建一个匹配的元素集合的深度拷贝副本。

Also in: CSS | Manipulation > Style Properties

.css()

获取匹配元素集合中的第一个元素的样式属性的值 或 设置每个匹配元素的一个或多个CSS属性。

Also in: Manipulation > DOM Removal

.detach()

从DOM中去掉所有匹配的元素。

Also in: Manipulation > DOM Removal

.empty()

从DOM中移除集合中匹配元素的所有子节点。

Also in: Attributes | Manipulation > Class Attribute | CSS

.hasClass()

确定任何一个匹配元素是否有被分配给定的（样式）类。

Also in: CSS | Dimensions | Manipulation > Style Properties

.height()

获取匹配元素集合中的第一个元素的当前计算高度值 或 设置每一个匹配元素的高度值。

Also in: Attributes | Manipulation > DOM Insertion, Inside

.html()

从集合的第一个匹配元素中获取HTML内容 或 设置每一个匹配元素的html内容。

Also in: CSS | Dimensions | Manipulation > Style Properties

.innerHeight()

为匹配的元素集合中获取第一个元素的当前计算高度值,包括padding，但是不包括border。

Also in: CSS | Dimensions | Manipulation > Style Properties

.innerWidth()

为匹配的元素集合中获取第一个元素的当前计算宽度值,包括padding，但是不包括border。

Also in: Manipulation > DOM Insertion, Outside

.insertAfter()

在目标元素后面插入集合中每个匹配的元素。

Also in: Manipulation > DOM Insertion, Outside

.insertBefore()

在目标元素前面插入集合中每个匹配的元素。

Also in: Manipulation > Style Properties

jQuery.cssNumber

一个对象，这个包含所有可以不使用单位的CSS属性。.css()方法使用这个对象来确定是否可能附加px到无单位的值。

jQuery.htmlPrefilter()

通过jQuery操作方法修改和过滤HTML字符串。

Also in: CSS | Offset | Manipulation > Style Properties

.offset()

在匹配的元素集合中，获取的第一个元素的当前坐标， 或设置每一个元素的坐标， 坐标相对于文档。

Also in: CSS | Dimensions | Manipulation > Style Properties

.outerHeight()

为匹配的元素集合中获取第一个元素的当前计算高度值,包括padding，border和选择性的margin。返回一个整数（不包含“px”）表示的值 或 null如果空集合的元素调用。

Also in: CSS | Dimensions | Manipulation > Style Properties

.outerWidth()

为匹配的元素集合中获取第一个元素的当前计算宽度值,包括padding，border。

Also in: CSS | Offset | Manipulation > Style Properties

.position()

获取匹配元素中第一个元素的当前坐标，相对于offset parent的坐标。( 译者注：offset parent指离该元素最近的而且被定位过的祖先元素 )

Also in: Manipulation > DOM Insertion, Inside

.prepend()

将参数内容插入到每个匹配元素的前面（元素内部）。

Also in: Manipulation > DOM Insertion, Inside

.prependTo()

将所有元素插入到目标前面（元素内）。

Also in: Attributes | Manipulation > General Attributes

.prop()

获取匹配的元素集中的第一个元素的属性（property）值或设置每一个匹配元素的一个或多个属性。

Also in: Manipulation > DOM Removal

.remove()

从DOM中去掉所有匹配的元素。

Also in: Attributes | Manipulation > General Attributes

.removeAttr()

为匹配的元素集合中的每个元素中移除一个属性。

Also in: Attributes | Manipulation > Class Attribute | CSS

.removeClass()

移除集合中每个匹配元素上一个，多个或全部样式。

Also in: Attributes | Manipulation > General Attributes

.removeProp()

为集合中匹配的元素删除一个属性（property）。

Also in: Manipulation > DOM Replacement

.replaceAll()

用集合的匹配元素替换每个目标元素。

Also in: Manipulation > DOM Replacement

.replaceWith()

用提供的内容替换集合中所有匹配的元素并且返回被删除元素的集合。

Also in: CSS | Offset | Manipulation > Style Properties

.scrollLeft()

获取匹配的元素集合中第一个元素的当前水平滚动条的位置或设置每个匹配元素的水平滚动条位置。

Also in: CSS | Offset | Manipulation > Style Properties

.scrollTop()

获取匹配的元素集合中第一个元素的当前垂直滚动条的位置或设置每个匹配元素的垂直滚动条位置。

Also in: Manipulation > DOM Insertion, Inside

.text()

得到匹配元素集合中每个元素的文本内容结合，包括他们的后代，或设置匹配元素集合中每个元素的文本内容为指定的文本内容。

Also in: Attributes | Manipulation > Class Attribute | CSS

.toggleClass()

在匹配的元素集合中的每个元素上添加或删除一个或多个样式类,取决于这个样式类是否存在或价值切换属性。即：如果存在（不存在）就删除（添加）一个类。

Also in: Manipulation > DOM Insertion, Around | Manipulation > DOM Removal

.unwrap()

将匹配元素集合的父级元素删除，保留自身（和兄弟元素，如果存在）在原来的位置。

Also in: Attributes | Forms | Manipulation > General Attributes

.val()

获取匹配的元素集合中第一个元素的当前值或设置匹配的元素集合中每个元素的值。

Also in: CSS | Dimensions | Manipulation > Style Properties

.width()

为匹配的元素集合中获取第一个元素的当前计算宽度值或给每个匹配的元素设置宽度。

Also in: Manipulation > DOM Insertion, Around

.wrap()

在每个匹配的元素外层包上一个html元素。

Also in: Manipulation > DOM Insertion, Around

.wrapAll()

在所有匹配元素外面包一层HTML结构。

Also in: Manipulation > DOM Insertion, Around

.wrapInner()

在匹配元素里的内容外包一层结构。
